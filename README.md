# Download util

## Usage
```sh
bin/downloader ./download.list ~/Downloads > downloads.out.list
```

## Install with pip

```sh
pip3 install --upgrade git+https://github.com/audriusrudalevicius/download-util
```

## Dev setup

```sh
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
pip install -e .
```
